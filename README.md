\#⏱ Stopwatch



A simple Python stopwatch to track elapsed time.



\## Features

\- Start, stop, reset, and quit commands

\- Tracks elapsed time

\- Shows final time when quitting



\## Example Run

⏱ Simple Stopwatch

Commands: 'start', 'stop', 'reset', 'quit'



Enter command: start

Stopwatch started...



Enter command: stop

Stopped at 4.32 seconds.



Enter command: quit

Final time: 4.32 seconds. Goodbye! 👋
